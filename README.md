# Telegram Login / Onboarding UX & UI Redesign

## Project Overview
This project focuses on improving the user experience and visual design of the Telegram Login / Onboarding screen.

## Task Details

### Task 1: UX Task – Problem Identification
Selected Screen: Telegram Login / Onboarding Screen

### Identified UX Issues
1. Lack of visual hierarchy
2. Confusing onboarding flow
3. Weak spacing and alignment

## Task 2: UI Task – Visual Redesign
The screen was redesigned with:
- Improved typography
- Better spacing and layout
- Modern color palette
- Enhanced visual hierarchy
- Cleaner onboarding experience

- Canva
